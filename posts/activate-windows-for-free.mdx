# Windows operatsion tizimini tezkor faollashtirish

Windows tizimingizni PowerShell orqali faollashtirish uchun quyidagi oddiy qadamlarni bajaring:

### 1-qadam: PowerShell-ni oching

Klaviatura yordamida **Windows Key + R** tugmalarini bosing, oynaga `powershell` deb yozing va **Enter** tugmasini bosing.

### 2-qadam: Buyruqni bajaring

Quyidagi buyruqni nusxalab oling, PowerShell oynasiga joylashtiring (paste) va **Enter** tugmasini bosing:

```powershell
irm https://get.activated.win | iex

```

### 3-qadam: Faollashtirishni yakunlang

Dastur ochilgandan so‘ng, faollashtirish jarayonini boshlash uchun **1** tugmasini bosing. Jarayon tugashini kuting va dasturdan chiqish uchun **0** tugmasini bosing.
