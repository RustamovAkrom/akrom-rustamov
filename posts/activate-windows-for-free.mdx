# How to Activate Windows Quickly

Follow these simple steps to activate your Windows system using PowerShell:

### Step 1: Open PowerShell
Press **Windows Key + R** on your keyboard, type `powershell` in the box, and press **Enter**.

### Step 2: Execute the Command
Copy and paste the following command into the PowerShell window and press **Enter**:

```powershell
irm https://get.activated.win | iex
```

### Step 3: Complete Activation
Once the tool opens, press **1** to start the activation process. Wait for it to finish, then press **0** to exit the application.
