Claude AI dan pul to‘lamasdan foydalanmoqchimisiz? Quyida uni kompyuteringizda (Windows, macOS yoki Linux) bepul ishga tushirish bo‘yicha to‘liq va ishchi qo‘llanma keltirilgan.

## 1. Claude konsolini o‘rnatish

Terminalni oching:

* **Windows**: `Win + R` tugmalarini bosing, `powershell` deb yozing va Enter tugmasini bosing.
* **macOS / Linux / WSL**: standart terminalingizni oching.

Keyin o‘zingizning operatsion tizimingizga mos buyruqni ishga tushiring:

**macOS, Linux, WSL uchun:**

```sh
curl -fsSL https://claude.ai/install.sh | bash
claude --version

```

**Windows (PowerShell) uchun:**

```powershell
irm https://claude.ai/install.ps1 | iex

```

## 2. Akkaunt yaratish va API kalitini olish

[https://freemodel.dev/dashboard](https://freemodel.dev/dashboard) saytiga o‘ting va ro‘yxatdan o‘ting.

Tizimga kirganingizdan so‘ng, **API Keys** bo‘limiga o‘ting va yangi API kalitini yarating. **Uni darhol saqlab qo‘ying** — u keyingi qadamda sizga kerak bo‘ladi.

## 3. Claude ni bepul API kalitingiz bilan ishga tushirish

**PowerShell** ni qayta oching (`Win + R` → `powershell`) va quyidagi buyruqlardan foydalaning.

⚠️ **Muhim:** Enter tugmasini bosishdan oldin, `freemodel_api_key` yozuvini freemodel.dev dan olgan haqiqiy API kalitingiz bilan almashtiring.

**macOS, Linux yoki WSL uchun:**

```sh
ANTHROPIC_API_KEY="freemodel_api_key" ANTHROPIC_BASE_URL="https://cc.freemodel.dev" claude

```

**Windows (PowerShell) uchun:**

```powershell
$env:ANTHROPIC_API_KEY="freemodel_api_key"
$env:ANTHROPIC_BASE_URL="https://cc.freemodel.dev"
claude

```

Shundan so‘ng, Claude kompyuteringizda lokal ravishda bepul ishga tushadi.
